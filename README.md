# 📱 Trabalho Mobile - ADSIS5S

Repositório do trabalho desenvolvido na disciplina de Desenvolvimento Mobile, utilizando **Dart e Flutter**.

## 📌 Descrição do Projeto

Este projeto tem como objetivo o desenvolvimento de uma aplicação mobile como parte do processo de aprendizagem da disciplina, aplicando conceitos fundamentais de Flutter, como:

- Estruturação de telas
- Gerenciamento de estado
- Implementação de funcionalidades básicas
- Organização de código em componentes

## 🚀 Tecnologias Utilizadas

- Flutter
- Dart
- VS Code
- Git & GitHub

## 📂 Estrutura do Projeto

O projeto está organizado de forma a facilitar a compreensão e evolução do código, separando responsabilidades entre interfaces, lógica e componentes reutilizáveis.
